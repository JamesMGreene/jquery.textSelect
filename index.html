jquery.textSelect
by JamesMGreene

jQuery plugin that creates a bindable special event for user text selection: "textSelect"

Have you ever wished that browsers threw an event when user select text on your page? If you are [or can be] using jQuery 1.7+, then this simple jQuery extension can grant that wish for you!

Usage

jQuery(selector).on("textSelect", callbackFunc);
jQuery(selector).textSelect(callbackFunc);

Dependencies

jQuery 1.7+
